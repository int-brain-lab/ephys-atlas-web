IBL Ephys Atlas

A brain atlas for exploring region-level electrophysiology features.

The IBL ephys atlas brings together anatomical mappings, slice views, summary plots, and 3D exploration to help users inspect how electrophysiology-derived features vary across brain regions. It supports atlas-based exploration of curated feature buckets, custom uploads, and multiple region mappings within a single interface.

Open the atlas Read the docs

Explore multiple views

Move between coronal, sagittal, horizontal, projection, summary, and 3D views while keeping a shared regional context.

Inspect feature buckets

Load built-in datasets or work with uploaded features to compare statistics, mappings, and color encodings across the atlas.

Bridge anatomy and data

Use the atlas as an entry point for understanding how electrophysiology features relate to anatomical structure at region level.
